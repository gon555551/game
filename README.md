# GAME !!!!

Currently has these controls:
- q   -> go up left
- w   -> go up
- e   -> go up right
- a   -> go left
- d   -> go right
- z   -> go down left
- x   -> go down
- c   -> go down right
- s   -> rest for 1 turn
- <   -> climb down stairs
- \>   -> climb up stairs
- p   -> display stats
- g   -> grab an item
- l   -> leave an item
- .   -> move the item selector
- i   -> display inventory
- ESC -> enable program exit
